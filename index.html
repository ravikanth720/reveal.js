Variable Scope


JavaScript has no block scope.



function printSomething( ) {

	console.log("The variable x = "+x);
	if( true ) {
		var x = "I'm declared inside if block";
		console.log(x);
	}

}
OUTPUT

The variable x = undefined
I'm declared inside if block


Only function level scope. Any variable declared with var is available throughout the function. when I print x before declaring it, it still has a value of undefined.

let keyword

let keyword allows block level scoping



function printSomething( ) {

	console.log("The variable x = "+x);
	if( true ) {
		let x = "I'm declared inside if block";
		console.log(x);
	}

}
OUTPUT

ReferenceError: x is not defined

The newly introduced let keyword allows block scoped variables This is helpful inside loops.

Changes in Functions


Arrow functions

A shorter syntax to declare functions


var multiply = (a, b) => a*b;

Usage 

multiply(5,4);

Older Syntax 

var multiply = function(a,b){
	return a*b;
}
here the anonymous function is assigned to multiply. With new arrow syntax, the values inside paranthesis are parameters and the return value comes after the fat arrow.

Arrow functions

Some examples


var nums = [1,2,3,4,5,6,7,8];

//filter out even numbers
var evens = nums.filter(x => x%2 == 0);

//generate odds from even array
var odds = evens.map(v => v+1);

Output 

> evens
> Array [ 2, 4, 6, 8 ]

> odds
> Array [ 3, 5, 7, 9 ]

Default Values for Parameters


function printSum(a=3, b=6){
	return a+b;
}


Output 

> printSum();
> 9

> printSum(5, 5);
> 10

Rest Parameters

This allows variable number of parameters to functions


function printSum(a, b, ...other){

	console.log("a ="+a+" b = "+b);
	console.log("other :")
	console.log(other);

}


Rest parameter 


> printSum(1, 2, 3, 4, 5, 6);
> a = 1 b = 2
> other:
> Array [ 3, 4, 5, 6 ]
Here the first two parameters are assigned to a, b the rest of the parameters come in as an array and it is assigned to rest

Spread Operator

Substitutes a single iterable object for multiple parameters


> var nums = [1,2,3,4,5,6,7,8];

//find minimum of the nums
> Math.min(...nums);
> 1

Old JavaScript 

> Math.min(nums);
> NaN

This gives our output
>Math.min(1, 2, 3, 4, 5, 6, 7, 8);
> 1

Also
> Math.min.apply(Math, nums);
> 1


Spread operator is similar to rest parameters except it is used while calling functions with multiple parameters.
